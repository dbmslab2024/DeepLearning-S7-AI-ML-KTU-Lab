7-line step-by-step algorithm for your Linear Regression with Gradient Descent program, suitable for a lab manual:

1.  **Generate Data:** Create synthetic linear data points `X` and `y` with added random noise.
2.  **Initialize Parameters:** Set initial values for the slope (`m`), intercept (`c`), learning rate (`alpha`), and number of `epochs`.
3.  **Gradient Descent Loop:** Iterate for the specified `epochs` to optimize parameters.
4.  **Calculate Prediction & Error:** Inside the loop, compute `y_pred` and the `error` (`y_pred - y`).
5.  **Compute Cost & Gradients:** Calculate the Mean Squared Error (`cost`) and the gradients (`dm`, `dc`).
6.  **Update Parameters:** Adjust `m` and `c` using the learning rate and calculated gradients.
7.  **Visualize Results:** Plot the final fitted regression line on the data and the cost function's reduction over epochs.
