10-line step-by-step algorithm for your SVM classification program, suitable for a lab manual:

1.  **Load Dataset:** Import and load the Iris dataset (or your chosen dataset).
2.  **Split Data:** Divide the dataset into training (80%) and testing (20%) sets.
3.  **Feature Scale:** Apply `StandardScaler` to normalize features for optimal SVM performance.
4.  **Initialize SVM:** Create an `SVC` classifier instance, specifying a `linear` kernel.
5.  **Train Model:** Fit the SVM model using the scaled training data and labels.
6.  **Make Predictions:** Use the trained model to predict labels for the scaled test data.
7.  **Calculate Accuracy:** Compute the `accuracy_score` by comparing predicted and true labels.
8.  **Generate Confusion Matrix:** Create the `confusion_matrix` to assess classification performance.
9.  **Generate Classification Report:** Print the `classification_report` for precision, recall, and F1-score.
10. **Visualize Results:** Display the confusion matrix using `seaborn.heatmap` for visual analysis.
